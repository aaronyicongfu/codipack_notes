# CoDiPack Notes: Get Started

Documentation and Tutorials can be found [here](https://www.scicomp.uni-kl.de/codi/).

## Usage
CoDiPack is a header-only c++ library. All need to do is to
```c++
#include <codi.cpp>
```
Code can be get using:
```
$ git clone https://github.com/SciCompKL/CoDiPack.git
```
